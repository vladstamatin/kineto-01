Component Specification: Navigation Bar

1. Layout & Behavior

   Type: Fixed Sticky Header.
   Background: Fully Transparent (0% Opacity) initially to overlay the hero video. Transitions to Solid White (#FFFFFF) with a subtle shadow box-shadow: 0 4px 20px rgba(0,0,0,0.05) upon scrolling.
   Desktop Grid: 3-Column Flexbox.
   Left (35%): Navigation Links (Horizontal).
   Center (30%): Brand Logo (Centered).
   Right (35%): Action Stack (Vertical Alignment).
   Mobile Grid:
   Left: Hamburger Menu (/// icon).
   Center: Brand Logo.
   Right: Action Stack (Vertical).

2. Branding (Centerpiece)

   Composition: Vertical Stack.
   Visual Elements:
   Top: Abstract "Hands" Icon (Using the Green/Blue gradient from the brand assets).
   Bottom: "REBALANCE" text (Deep Medical Navy #1F2A38, Uppercase, Clean Sans-Serif).
   Sizing: Height constraint of 70-80px to maintain a sleek profile.

3. Navigation Links (Left Side)

   Items: Sus, servicii si preturi, echipa, contacte.
   Desktop Styling: Horizontal row. Font: Manrope or Plus Jakarta Sans, Weight: 500. Color: Slate Grey (high contrast against video or white).
   Mobile Interaction: "Hamburger" icon (three thin lines) triggers a slide-out drawer from the left containing the menu items.

4. Action Components (Right Side)

   Container Layout: Vertical Flex Column (Align Right). Elements are stacked to conserve width.
   Row 1: Phone Contact Button (Primary CTA)
   Text: "060123456"
   Style: Use a hand icon with a phone icon combined, Solid White fill, Thin Turquoise border (#00C4CC).
   Icon Detail: Be creative and give an interesting and easy way to represent phone button following design direction.
   Row 2: Language Switcher
   Structure: Inline text links: RO | RU | EN
   Typography: Micro-label size (11px-12px), Uppercase.
   State: Selected language in Bold Navy; inactive languages in Grey.

5. Technical Implementation Notes

   Z-Index: 1000 (Highest priority layer).
   Responsiveness:
   Desktop: Full 3-column layout.
   Tablet/Mobile: Navigation links hide behind the Hamburger menu. The Right Side "Action Stack" (Phone + Lang) remains visible but scales down slightly to fit the narrower viewport.
